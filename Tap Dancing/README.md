# Tap Dancing

Kode angka tersebut adalah morse code. 1 melambangkan - atau _ 2 melambangkan . 0 melambangkan (spasi) Kode angka tersebut diubah menjadi kode morse
```
-- ----- .-. ... . -. ----- - -... ....- ... . ...--
```
Dengan bantuan morse translator, didapakan flagnya
